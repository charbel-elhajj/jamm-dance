# JAMM Dance Showcase Website

## Setup

```bash
npm install
```

## Commands

| Command          | Action                  |
| ---------------- | ----------------------- |
| `npm run dev`    | Start local dev server  |
| `npm run build`  | Build for production    |
| `npm run preview`| Preview production build|
